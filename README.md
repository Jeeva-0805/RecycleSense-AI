# ♻️ RecycleSense AI

An AI-powered waste management platform that helps users identify waste materials, understand environmental impact, and receive recycling guidance.

## Features

- AI Waste Detection
- Waste Category Classification
- AI Confidence Score
- Environmental Impact Analysis
- Waste Analytics Dashboard
- Detection History Tracking
- Sustainability Report Generator
- AI Recycling Assistant
- E-Waste Advisor

## Technologies Used

- Python
- Streamlit
- TensorFlow
- Pandas
- MobileNetV2
- CSV Data Storage

## SDG Alignment

- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production
- SDG 13: Climate Action
